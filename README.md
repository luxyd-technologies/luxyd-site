# Luxyd Static Site

Static marketing site for Luxyd Technologies.

## Publish on GitHub Pages
1. Push this repo to GitHub.
2. In **Settings → Pages**, set **Source** to the default branch and **/ (root)**.
3. Save and wait for the site to deploy.

## Local preview
Open `luxyd-site/index.html` in a browser.

## Structure
- `luxyd-site/` — HTML/CSS/JS and assets for the site.
